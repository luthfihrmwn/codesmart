## ðŸŽ¨ Color Usage Guidelines

### Primary Actions
- **Purple (#754ef9)**: Main CTAs, primary buttons, links
- **Purple Gradient**: Hero sections, cards, important elements

### Status Colors
- **Green (#00d25b)**: Success, approved, completed
- **Red (#e74c3c)**: Danger, rejected, error
- **Orange (#f39c12)**: Warning, pending
- **Blue (#0ea5e9)**: Info, notifications
- **Yellow (#ffc107)**: Caution, incomplete

### Neutral Colors
- **Dark (#333)**: Body text
- **Gray (#666)**: Secondary text
- **Light Gray (#f8f9fa)**: Backgrounds
- **White (#fdfdfd)**: Cards, modals

### Subtle Animations
- Button hover: translateY(-3px)
- Card hover: translateY(-5px)

### Page Transitions
- Fade in: 0.6s
- Slide in: 0.6s from side
- Modal: scale from 0.9 to 1
